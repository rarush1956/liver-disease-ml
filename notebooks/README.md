## Notebook Description

### `liver_disease_model.ipynb`

This notebook includes the complete experimental workflow:

- Loading and inspecting the Indian Liver Patient Dataset (ILPD)
- Handling missing values and encoding categorical features
- Feature scaling using standardization
- Handling class imbalance using SMOTE oversampling
- Training and comparing multiple machine learning models
- Evaluating models using accuracy, confusion matrix, and ROC curve
- Interpreting results and identifying key predictive features

The notebook was primarily used during the research and development
phase before finalizing the model architecture.
